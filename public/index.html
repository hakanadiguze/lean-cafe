How it works

Four simple phases, everyone participates

1
✍️ Write Notes
Everyone writes sticky notes about what they want to discuss — one topic per note. All notes appear on screen in real time for everyone to see.
2
🗳️ Vote
Each person gets a set number of votes. Vote for the notes that matter most to you. The top-voted notes become discussion rooms.
3
💡 Discuss
The facilitator creates discussion rooms from the top-voted notes. Groups discuss in parallel (e.g. in Teams breakout rooms). One person per group — the Scribe — enters the root cause and action items.
4
📄 Report
When all groups are done, the facilitator ends the session. A full summary with all notes and action items can be downloaded as a PDF.
